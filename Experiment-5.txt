Infix To Postfix
Problem Statement:

Objective(s): To learn the Shunting-yard algorithm for converting infix expressions to postfix (RPN) and to implement it using a stack.



Outcome: Students will be able to write a program that translates a standard infix

mathematical expression into its postfix equivalent, a crucial step for computer evaluation.



Problem Statement: Write a program for a given infix expression, and convert it to postfix

notation.



Background Study: Infix notation (e.g., a + b * c) is ambiguous without operator

precedence rules. Postfix notation (a b c * +) is unambiguous and easily evaluated with a

stack. The Shunting-yard algorithm converts infix to postfix.

It uses a stack to hold operators.

1. Operand: Read an operand, append it to the output string.

2. (: Read (, push it onto the operator stack.

3. ): Read ), pop operators from the stack and append to output, until a ( is seen. Pop

and discard the (.

4. Operator: Read an operator op. While the stack is not empty, and the top is not (,

and the top has precedence greater than or equal to op, pop from stack to output.

Then, push op onto the stack.

5. End: When the input is finished, pop all remaining operators from the stack to the

output.

Input Explanation:

A single line containing a valid infix expression.

Example:

a+b*c

Output Explanation:

Postfix (RPN) expression.

Example:

abc*+

Constraints:

The expression may contain:

- Operands: a–z, A–Z, 0–9

- Operators: +, -, *, /, ^

- Parentheses: ( )

Length of expression ≤ 1000

Time Limit:
1 sec

Example 1:
Input
a+b*c

Output
abc*+

Example 2:
Input
(a+b)*c

Output
ab+c*

Sample I/O Explanation:

Symbol Action
a	Operand → output = a
+	Push +
b	Operand → output = ab
*	Higher precedence than + → push *
c	Operand → output = abc
End	Pop *, then +


====================================================================================================================================================================

import java.util.*;
public class InfixToPostfix {
    // Function to return precedence of operators
    static int precedence(char c) {
        if (c == '^') return 3;
        if (c == '*' || c == '/') return 2;
        if (c == '+' || c == '-') return 1;
        return 0;
    }
    // Function to convert infix to postfix
    public static String infixToPostfix(String exp) {
        StringBuilder output = new StringBuilder();
        Stack<Character> stack = new Stack<>();

        for (int i = 0; i < exp.length(); i++) {
            char c = exp.charAt(i);

            // Operand: append to output
            if (Character.isLetterOrDigit(c)) {
                output.append(c);
            }
            // '(': push onto stack
            else if (c == '(') {
                stack.push(c);
            }
            // ')': pop until '(' is found
            else if (c == ')') {
                while (!stack.isEmpty() && stack.peek() != '(') {
                    output.append(stack.pop());
                }
                stack.pop(); // discard '('
            }
            // Operator
            else {
                while (!stack.isEmpty() && stack.peek() != '(' &&
                       precedence(stack.peek()) >= precedence(c)) {
                    output.append(stack.pop());
                }
                stack.push(c);
            }
        }

        // Pop all remaining operators
        while (!stack.isEmpty()) {
            output.append(stack.pop());
        }

        return output.toString();
    }
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String exp = sc.nextLine();
        System.out.println(infixToPostfix(exp));
        sc.close();
    }
}
