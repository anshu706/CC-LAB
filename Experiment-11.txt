Build a Binary Search Tree (BST)
Problem Statement:

Objective(s)
To understand the insertion operation in a Binary Search Tree (BST) and implement a function insert() that correctly inserts new values while maintaining BST properties.

Outcome
Students will be able to:

Understand the fundamental insertion process in a BST
Implement a recursive BST insertion algorithm
Build a BST from a sequence of input values
Verify the correctness of the BST using in-order traversal
Understand how BST properties are preserved during insertion
Problem Statement
Write a program to construct a Binary Search Tree (BST) by implementing the insert() function and verify the tree using in-order traversal.

Input Explanation:

n

v1 v2 v3 … vn

Where:

n = number of elements
v1, v2, … vn = values to be inserted into the BST in the given order


Output Explanation:

Print the in-order traversal of the constructed BST.

If the BST is built correctly, the output will be in sorted order.

Constraints:

Number of nodes: 0 ≤ n ≤ 10⁵

Node values: any integers

Duplicate values are ignored

Insertion is done one by one

Time Limit:
1 sec

Example 1:
Input
9
8 3 10 1 6 14 4 7 13

Output
1 3 4 6 7 8 10 13 14

Example 2:
Input
5
1 2 3 4 5

Output
1 2 3 4 5

Sample I/O Explanation:

The values are inserted one by one into the BST

In-order traversal visits nodes in the order:

Left → Root → Right

Since BST properties are maintained, the output is sorted


==============================================================================================================================


import java.util.Scanner;

class TreeNode {
    int val;
    TreeNode left, right;
    TreeNode(int val) {
        this.val = val;
        left = right = null;
    }
}

public class BuildBST {
    public static TreeNode insert(TreeNode root, int val) {
        if (root == null) {
            return new TreeNode(val);
        }
        if (val < root.val) {
            root.left = insert(root.left, val);
        } else if (val > root.val) {
            root.right = insert(root.right, val);
        }
        // Duplicate values are ignored (val == root.val does nothing)
        return root;
    }

    public static void inOrderTraversal(TreeNode root) {
        if (root == null) return;
        inOrderTraversal(root.left);
        System.out.print(root.val + " ");
        inOrderTraversal(root.right);
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        TreeNode root = null;
        if (n > 0) {
            for (int i = 0; i < n; i++) {
                int val = sc.nextInt();
                root = insert(root, val);
            }
        }
        inOrderTraversal(root);
        System.out.println();
        sc.close();
    }
}
