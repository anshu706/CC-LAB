Design a circular queue
Problem Statement:

Objective(s): To understand the concept and implementation of a circular queue (circular

buffer), a fixed-size queue that efficiently uses its storage.



Outcome: Students will be able to design and implement a fixed-size circular queue

supporting enQueue, deQueue, Front, and Rear operations, and correctly handle empty and full (overflow/underflow) conditions.



Problem Statement: Write a program to design a circular queue of size k which should

implement the below functions: a. enQueue(value): Adds an item to the queue. Returns true

on success. b. deQueue(): Removes an item from the queue. Returns true on success. c.

Front(): Get the front item from the queue. d. Rear(): Get the last item from the queue. e.

isEmpty() / isFull()



Background Study: A standard queue implemented with an array can waste space. When

elements are dequeued, the space at the front is left empty. A circular queue solves this by

treating the array as circular. The position after the last element is the first element.

We use two pointers, head (or front) and tail (or rear).

● head points to the index of the first element.

● tail points to the index of the last element.

● enQueue: Insert at (tail + 1) % size.

● deQueue: Remove from head, then move head to (head + 1) % size.

Managing empty/full states can be tricky. A common method is to store the size (capacity)

and a count of current elements.

● Empty: count == 0

● Full: count == size This avoids the ambiguity of head == tail.

import java.util.*;

public class CircularQueueProgram {

    static class MyCircularQueue {
        private int[] queue;
        private int head;
        private int tail;
        private int size;
        private int count;

        // Constructor
        public MyCircularQueue(int k) {
            queue = new int[k];
            size = k;
            head = 0;
            tail = -1;
            count = 0;
        }

        // Insert element into circular queue
        public boolean enQueue(int value) {
            if (isFull()) {
                return false;
            }
            tail = (tail + 1) % size;
            queue[tail] = value;
            count++;
            return true;
        }

        // Delete element from circular queue
        public boolean deQueue() {
            if (isEmpty()) {
                return false;
            }
            head = (head + 1) % size;
            count--;
            return true;
        }

        // Get front element
        public int Front() {
            if (isEmpty()) {
                return -1;
            }
            return queue[head];
        }

        // Get rear element
        public int Rear() {
            if (isEmpty()) {
                return -1;
            }
            return queue[tail];
        }

        // Check if queue is empty
        public boolean isEmpty() {
            return count == 0;
        }

        // Check if queue is full
        public boolean isFull() {
            return count == size;
        }
    }

    // Main program
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int k = sc.nextInt();  // size of queue
        int Q = sc.nextInt();  // number of operations

        MyCircularQueue cq = new MyCircularQueue(k);

        while (Q-- > 0) {
            int op = sc.nextInt();

            switch (op) {
                case 1:
                    int value = sc.nextInt();
                    System.out.println(cq.enQueue(value));
                    break;
                case 2:
                    System.out.println(cq.deQueue());
                    break;
                case 3:
                    System.out.println(cq.Front());
                    break;
                case 4:
                    System.out.println(cq.Rear());
                    break;
                case 5:
                    System.out.println(cq.isEmpty());
                    break;
                case 6:
                    System.out.println(cq.isFull());
                    break;
                default:
                    System.out.println("Invalid Operation");
            }
        }
        sc.close();
    }
}
