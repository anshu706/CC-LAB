Merge two sorted linked lists
Problem Statement:

Objective(s): To understand the structure of linked lists and implement the merge operation for two sorted linked lists to create a single sorted list.



Outcome: Students will be able to manipulate object references in linked lists and

implement an efficient in-place merge algorithm.



Problem Statement: Write a program to merge two sorted linked lists into a single sorted

linked list.



Background Study: Merging two sorted lists is a fundamental linked list operation. The goal

is to combine the nodes of two lists (which are already sorted) into a single sorted list.

The iterative approach is efficient and uses O(1) space. It involves:

1. Creating a dummy head node. This simplifies the logic, as we don't need a special

case for the head of the new list.

2. Maintaining a tail pointer that always points to the last node in the merged list so far.

3. Comparing the heads of the two input lists (list1 and list2).

4. Appending the smaller of the two nodes to tail.next, and advancing the pointer of the

list from which the node was taken.

5. Moving tail to the node we just added.

6. When one list is exhausted, append the remainder of the other list to tail.next.

Input Explanation:

n

elements of list1 in sorted order

m

elements of list2 in sorted order

Output Explanation:

Merged sorted linked list elements

Constraints:

1 ≤ n, m ≤ 10^5
Linked list values may be negative or positive
Lists are sorted in ascending order
Time Limit:
1 sec

Example 1:
Input
3
1 3 5
3
2 4 6

Output
1 2 3 4 5 6

Example 2:
Input
4
1 2 7 10
2
3 6

Output
1 2 3 6 7 10

Sample I/O Explanation:

list1 list2 merged list
1	2	1
3	2	1 → 2
3	4	1 → 2 → 3
5	4	1 → 2 → 3 → 4
5	6	1 → 2 → 3 → 4 → 5
—	6	1 → 2 → 3 → 4 → 5 → 6

import java.util.*;

public class MergeTwoSortedLists {

    // Definition for singly-linked list.
    static class ListNode {
        int val;
        ListNode next;

        ListNode(int val) {
            this.val = val;
        }
    }

    // Function to merge two sorted linked lists
    public static ListNode mergeTwoLists(ListNode list1, ListNode list2) {
       //Write Your Code Here
       ListNode dummy = new ListNode(-1);
       ListNode curr = dummy;
       ListNode t1 = list1;
       ListNode t2 = list2;
       
       while(t1 != null && t2 != null)
       {
           if(t1.val < t2.val)
           {
               curr.next = t1;
               t1 = t1.next;
           }
           else
           {
               curr.next = t2;
               t2 = t2.next;
           }
           curr = curr.next;
       
        }
           if(t1!=null)
           {
               curr.next = t1;
           }
           if(t2!=null)
           {
               curr.next = t2;
           }
                return dummy.next;
       
    }

    //No Need To Edit This Part :-  Helper to create a list from an array
    public static ListNode createList(int[] arr) {
        ListNode dummy = new ListNode(0);
        ListNode curr = dummy;
        for (int x : arr) {
            curr.next = new ListNode(x);
            curr = curr.next;
        }
        return dummy.next;
    }

    // Print list
    public static void printList(ListNode head) {
        while (head != null) {
            System.out.print(head.val + " ");
            head = head.next;
        }
        System.out.println();
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // Input first list
        int n1 = sc.nextInt();
        int[] arr1 = new int[n1];
        for (int i = 0; i < n1; i++) arr1[i] = sc.nextInt();

        // Input second list
        int n2 = sc.nextInt();
        int[] arr2 = new int[n2];
        for (int i = 0; i < n2; i++) arr2[i] = sc.nextInt();

        ListNode list1 = createList(arr1);
        ListNode list2 = createList(arr2);

        ListNode merged = mergeTwoLists(list1, list2);

        printList(merged);
    }
}
