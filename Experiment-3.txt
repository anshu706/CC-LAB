Next Greater Element
Problem Statement:

Objective(s): To learn and implement an efficient stack-based algorithm to find the Next

Greater Element (NGE) for each element in an array.



Outcome: Students will be able to solve problems related to finding the next greater/smaller number

element in a sequence and understand the &quot;monotonic stack&quot; pattern.



Problem Statement: Write a program for finding the Next Greater Element (NGE) for each

element in a given array. The NGE for an element x is the first greater element on the right

side of x in the array. If no such element exists, the NGE is considered -1.



Background Study: A naive approach uses two nested loops: for each element, scan the

rest of the array to its right. This is O(n^2) and too slow for competitive programming.

A more efficient O(n) approach uses a stack. This is a classic & " monotonic stack; problem.



We process the array from right to left. The stack will store elements we have already seen,

maintaining a decreasing order. When processing an element arr[i]:

1. While the stack is not empty and the element at the top (stack.peek()) is less than or

equal to arr[i], pop it. These popped elements can never be the NGE for arr[i].

2. If the stack is now empty, it means no element to the right is greater. The NGE for

arr[i] is -1.

3. If the stack is not empty, stack.peek() is the first greater element to the right. This is

the NGE.

4. Push arr[i] onto the stack for future elements (to its left) to compare against.

Input Explanation:

First line: Integer n (number of elements)
Second line: n space-separated integers


Output Explanation:

Print the Next Greater Element for each array element in the same order, separated by spaces.

Constraints:

1 ≤ n ≤ 10⁵
Array elements may be positive or negative
Time Limit:
1 sec

Example 1:
Input
4
4 5 2 25

Output
5 25 25 -1

Example 2:
Input
5
13 7 6 12 10

Output
-1 12 12 -1 -1

Sample I/O Explanation:

Input:-

4

4 5 2 25



Output:-

5 25 25 -1



Explanation:-

NGE of 4 → 5

NGE of 5 → 25

NGE of 2 → 25

NGE of 25 → -1

==============================================================================================================================================

import java.util.*;

public class Main {

    public static int[] nextGreaterElement(int[] arr) {
        int n = arr.length;
        int[] nge = new int[n];
        Stack<Integer> stack = new Stack<>();

        // Traverse from right to left
        for (int i = n - 1; i >= 0; i--) {

            // Remove elements smaller than or equal to current element
            while (!stack.isEmpty() && stack.peek() <= arr[i]) {
                stack.pop();
            }

            // If stack is empty, no greater element exists
            if (stack.isEmpty()) {
                nge[i] = -1;
            } else {
                nge[i] = stack.peek();
            }

            // Push current element to stack
            stack.push(arr[i]);
        }

        return nge;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int[] arr = new int[n];

        for (int i = 0; i < n; i++) {
            arr[i] = sc.nextInt();
        }

        int[] result = nextGreaterElement(arr);

        for (int x : result) {
            System.out.print(x + " ");
        }
    }
}
