Design A Stack
Problem Statement:

Objective(s): To design and implement a specialized stack, MinStack, that supports all

standard stack operations (push, pop, top) along with an efficient getMin() operation in

constant time.



Outcome: Students will be able to augment a standard data structure with additional

functionalities and understand the time-space complexity trade-offs involved in designing

efficient algorithms.



Problem Statement: Write a program for implementing a MINSTACK which should support

operations like push, pop, top, and getMin in O(1) time. The stack should handle overflow

and underflow conditions (or in Java&#39;s case, throw exceptions for underflow).

1. Construct a stack.

2. Push elements.

3. Pop elements.

4. Get the Top element.

5. Retrieve the minimum element from the stack.



Background Study: In Java, the modern and preferred way to implement a stack is using

the Deque interface with its ArrayDeque implementation. A standard Deque provides push(),

pop(), and peek() in O(1) time. However, finding the minimum element would require

iterating through all elements, taking O(n) time.

To optimize this, we use an auxiliary Deque, let&#39;s call it minStack. This stack will store the

minimum element seen so far at each stage.

● When we push(x) onto the main stack, we compare x with the top of minStack. We

push Math.min(x, minStack.peek()) onto minStack.

● When we pop() from the main stack, we also pop() from minStack. This ensures that

minStack.peek() always holds the minimum element of the entire main stack in O(1)

time.

operation_2
Output Explanation:

Print value for getMin()
For invalid pop/top/getMin on empty stack → print "Stack Underflow"

−10^9 ≤ x ≤ +10^9

import java.util.*;

public class MinStackProgram {

    static class MinStack {
        Deque<Integer> mainStack = new ArrayDeque<>();
        Deque<Integer> minStack = new ArrayDeque<>();

        // Push operation
        public void push(int x) {
            mainStack.push(x);

            if (minStack.isEmpty()) {
                minStack.push(x);
            } else {
                minStack.push(Math.min(x, minStack.peek()));
            }
        }

        // Pop operation
        public void pop() {
            if (mainStack.isEmpty()) {
                System.out.println("Stack Underflow");
                return;
            }
            mainStack.pop();
            minStack.pop();
        }

        // Top operation
        public void top() {
            if (mainStack.isEmpty()) {
                System.out.println("Stack Underflow");
                return;
            }
            System.out.println(mainStack.peek());
        }

        // Get Minimum in O(1)
        public void getMin() {
            if (minStack.isEmpty()) {
                System.out.println("Stack Underflow");
                return;
            }
            System.out.println(minStack.peek());
        }
    }

    // Main Program to read input and run operations
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int Q = sc.nextInt();   // number of operations
        MinStack ms = new MinStack();

        while (Q-- > 0) {
            int op = sc.nextInt();

            switch (op) {
                case 1: // push
                    int x = sc.nextInt();
                    ms.push(x);
                    break;

                case 2: // pop
                    ms.pop();
                    break;

                case 3: // top
                    ms.top();
                    break;

                case 4: // getMin
                    ms.getMin();
                    break;

                default:
                    System.out.println("Invalid Operation");
            }
        }

        sc.close();
    }
}
